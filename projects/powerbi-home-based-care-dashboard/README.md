# 📊 Home-Based Care Dashboard (Power BI)

## 📌 Overview
This dashboard monitors Home-Based Care (HBC) program activities across Ukraine, focusing on beneficiaries, regional distribution, and operational trends.

It helps stakeholders track program performance and make data-driven decisions.

---

## 🎯 Objectives
- Track number of beneficiaries
- Analyze regional distribution
- Monitor trends over time
- Evaluate program coverage
- Review partner support across regions

---

## 🛠 Tools & Technologies
- Power BI (DAX, Data Modeling)
- Data Cleaning & Transformation
- Data Visualization

---

## 📈 Key Features
- KPI metrics (beneficiaries, social helpers, locations)
- Interactive regional map
- Trend analysis over time
- Partner contribution analysis
- Filters by region and partner

---

## 🚀 Insights
- Program reach expanded significantly over time
- Service distribution varies by region
- Partner support plays a major role in coverage and delivery
