# 📊 Mobile Health Units Dashboard (Power BI)

## 📌 Overview
This dashboard analyzes the performance of Mobile Health Units (MHU) across Ukraine, focusing on consultations, visits, regional coverage, and partner contributions.

It provides a clear view of healthcare service delivery and helps stakeholders monitor program scale and efficiency.

---

## 🎯 Objectives
- Track number of consultations and visits
- Analyze regional distribution of mobile units
- Evaluate partner contributions
- Monitor program growth over time
- Understand beneficiary demographics

---

## 🛠 Tools & Technologies
- Power BI (Data Modeling, DAX)
- Data Cleaning & Transformation
- Data Visualization

---

## 📈 Key Features
- KPI metrics (consultations, visits, partners)
- Interactive map of Ukraine (regional analysis)
- Time-based trend analysis
- Partner performance breakdown
- Demographic insights (age & gender)

---

## 🚀 Insights
- Strong growth in healthcare services from 2022 to 2023
- Uneven regional distribution of mobile units
- Key partners responsible for majority of operations
- Clear patterns in beneficiary demographics
